

## Description

It's a simple crud backend application developed on nest js.

Just used a simple pipes validators to handle data.

connected to mongodb atlas.

[Nest](https://github.com/nestjs/nest) 


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test simple app

to test the simple backend application.

you just need to run the project and use a REST client (eg postman: https://www.postman.com/) to make API requests.

For the example, precisely, it is "localhost:3000/room"

you can interact with the API and get the responses from the different methods {'POST', 'GET', 'UPDATE', 'DELETE'}

it is not a production application but it complies with the principles of a backend
